# gestion-parc-auto
interface web qui permet de gérer sa flotte de véhicules

pour l'utilisé vous aurez trois étapes à faire : 

  - Importer le fichier "voiture.sql" dans votre base de données
  - Modifier les valeurs "$user", "$password", "dbname" & "$servername" dans le fichier "db.php" qui se trouve dans le dossier "inc"
  - Pour créer un utilisateur, modifier les valeurs "$login" & "$password" dans le fichier "create_user.php"

vous pouvez maintenant commencé à utilisé l'interface par vous mêmes
